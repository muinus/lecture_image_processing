# 課題2レポート

フリー素材アイドル「MIKA☆RIKA」 http://mika-rika-free.jp/ よりダウンロードしたフリー画像を原画像とする．
この画像は縦544画像，横827画素による正方形のディジタルカラー画像である．

ORG=imread('ex1.png'); % 原画像の入力  
imagesc(ORG); axis image; % 画像の表示
